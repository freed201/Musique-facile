# Lexique et tournures : liste de référence

Ouvrir ce fichier pour une vérification fine ou un doute sur un mot précis. Le SKILL.md contient déjà les marqueurs les plus fréquents ; ce fichier est la version exhaustive.

Rappel : aucun mot n'est interdit dans l'absolu. Un mot de cette liste est suspect quand il est **vide de sens dans son contexte** ou **accumulé** avec d'autres marqueurs. Le test : « est-ce que ce mot apporte une information que le lecteur n'aurait pas sans lui ? » Si non, supprimer ou préciser.

## 1. Mots et adjectifs surreprésentés

| Terme | Pourquoi ça trahit | Remplacement |
| --- | --- | --- |
| véritable | amplificateur creux | supprimer, ou préciser |
| incontournable | passe-partout marketing | utile, très utilisé, difficile à éviter |
| crucial | dramatisation | important, clé, ou supprimer |
| essentiel | dramatisation | important, ou supprimer |
| vital | dramatisation | important (rarement justifié) |
| robuste | fourre-tout technique | fiable, qui tient en production, solide |
| riche | évalue sans décrire | varié, détaillé, avec beaucoup d'exemples |
| fascinant / passionnant | enthousiasme générique | dire pourquoi c'est intéressant |
| dynamique | flou positif | actif, qui bouge, en croissance |
| nuancé | signal lexical IA | pas si simple, à prendre avec prudence |
| fluide / sans couture | calque (seamless) | simple, sans friction, qui s'enchaîne bien |
| pertinent | évaluation vague | utile ici, adapté, qui répond au besoin |
| innovant | marketing creux | nouveau, inédit, ou décrire ce qui change |
| puissant | amplificateur | efficace, ou décrire l'effet |
| optimiser | jargon | améliorer, accélérer, réduire (ce qui est concret) |
| notamment (en excès) | remplissage | par exemple, ou supprimer |
| de surcroît | connecteur soutenu IA | en plus, et |
| paysage (numérique, médiatique...) | calque landscape | secteur, marché, web |

| --- | --- |
| permettre de | aider à, éviter de, débloquer, faire gagner, donner accès à |

| mettre en avant | montrer, prouver, rappeler, mettre en avant un fait précis |
| approfondir | creuser, détailler, aller plus loin |
| explorer (en excès) | regarder, tester, voir |
| s'inscrire dans | faire partie de, aller avec |
| accompagner (jargon) | aider, suivre, former |

## 3. Tournures d'ouverture et de transition

| Tournure | Remplacement |
| --- | --- |
| À l'ère de… | Aujourd'hui, depuis que…, avec… |
| À l'heure où… | Depuis quelques mois…, maintenant que… |
| Dans un monde où… | commencer par un fait concret |
| Dans le paysage de… | dans le secteur, sur le marché |
| Que vous soyez X, Y ou Z… | cibler une seule situation : « Si tu débutes… » |
| Il convient de noter que | À noter, ou rien |
| Il est important de | Il faut, le point clé :, ou rien |
| Force est de constater que | On voit bien que |
| La question est donc la suivante : | poser la question directement |
| Pourquoi est-ce important ? | Le problème :, l'intérêt : |
| Mais ce n'est pas tout. | enchaîner directement |
| Plongeons dans… | regardons, voyons |

## 4. Tournures de clôture

| Tournure | Remplacement |
| --- | --- |
| En somme | Bref, au fond, ou rien |
| En définitive | à retenir :, ou rien |
| Pour conclure | terminer sur une phrase concrète |
| En résumé (systématique) | supprimer si le texte est court |
| N'hésitez pas à… | une formule plus directe selon le contexte |
| En espérant que… | formule de politesse normale |

## 5. Hedging (prudence excessive)

L'IA empile les précautions. Elles diluent le propos.

| Hedging | Action |
| --- | --- |
| il est important de noter que | supprimer, garder l'info |
| on pourrait dire que | dire les choses directement |
| dans une certaine mesure | supprimer ou chiffrer |
| il est possible que / il se peut que | si c'est vrai, l'affirmer ; sinon préciser la condition |
| généralement, souvent, parfois (empilés) | en garder un seul |

## 7. Marqueurs de mise en forme

| Marqueur | À faire à la place |

## Le test final en une phrase

Si on remplace le sujet du texte par n'importe quel autre sujet et que les phrases tiennent toujours, c'est que le texte ne dit rien. Ajouter du concret : un chiffre, un exemple, un cas, un détail vécu.
