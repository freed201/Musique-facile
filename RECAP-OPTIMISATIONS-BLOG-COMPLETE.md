### Priorisation Intelligente

## 🐛 Problèmes Potentiels & Solutions

### "La page /blog ne s'affiche plus"
**Cause probable :** Erreur de syntaxe dans le JSX/Astro
**Solution :** Vérifier les logs `npm run dev`, corriger la syntaxe

### "L'article optimisé casse le build"
**Cause probable :** Frontmatter invalide (guillemets, YAML)
**Solution :** Valider le YAML avec un linter

### "Les liens internes sont cassés"
**Cause probable :** Slug incorrect ou article non publié
**Solution :** Vérifier que les articles existent avec prod: Y

### "Le style CSS ne s'applique pas"
**Cause probable :** Classe CSS manquante
**Solution :** Vérifier les classes dans ArticleLayout.astro

### "Dois-je remplacer l'article original par l'optimisé ?"
**Réponse :** Teste d'abord en local. Si validé, remplace. Garde une copie de l'ancien dans /Old/ si besoin.

### "Combien d'articles dois-je optimiser par semaine ?"
**Réponse :** 3-5 articles/semaine est un bon rythme (15-20h de travail). Qualité > Quantité.

### "Faut-il optimiser les tutoriels de chansons ?"
**Réponse :** Oui, mais simplifié. Focus sur : titre optimisé, intro directe, FAQ 3 questions minimum.
